## 初始化本地库 ：        $ git init
## 克隆远端库 ：        $ git clone 分支名
## 查看远程库地址 ：    $ git remote -v
## 添加远程库地址 ：    $ git remote 别名 地址
## 删除远程库地址 ：    $ git remote remove 别名
## 加入暂存区 ：        $ git add
## 提交 ：              $ git commit -m "注释"
## 上传远程库 ：        $ git push 别名 分支名
## 拉取远程库 ：        $ git pull 别名 分支名


## 注：ssh方式可以不用登录，但需要ssh公钥
